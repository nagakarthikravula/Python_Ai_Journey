## Week 1 Project — Contact Book CLI App

A command-line contact management app that runs in the terminal.

**What it does:**
- Add a contact with name, phone number, and email
- View all saved contacts
- Search for a contact by name
- Delete a contact
- Contacts are stored in a nested dictionary structure

**How to run:**
```
python contact_book.py
```

**Functions:**
- `display_all_contacts()` — shows all saved contacts
- `add_contacts()` — adds a new contact
- `search_contacts()` — searches for a contact by name
- `delete_contact()` — removes a contact

**What I learned building this:**
- How to structure a CLI app with a menu loop
- How to use nested dictionaries to store structured data
- How to handle invalid user input gracefully
- The difference between functions that print vs functions that return
